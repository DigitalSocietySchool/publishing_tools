• Citations

This module will provide the journalist with an automatic way of setting up their bibliographies without having to type them manually.


How we can implement it in our tool:
In the editor section, when the journalist decides to use a given source the module will automatically create a bibliography with such source mentioned.


What’s the benefit for the journalist?
It will speed up their writing process because they won’t have to care about looking for the sources and following the standards of citation when set up their bibliographies.
